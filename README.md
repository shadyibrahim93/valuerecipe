# Next.js Recipe App (Prototype)

## Quickstart

1. Copy files into a project folder.
2. `npm install`
3. Create a Supabase project (free tier) and run the SQL in `db/schema.sql` or use the `/scripts/seed.js`.
4. Create `.env.local` from `.env.example` and paste Supabase values.
5. Run `npm run seed` to insert dummy data.
6. Run `npm run dev` and open http://localhost:3000

## Features

- Recipes with SEO-friendly JSON-LD
- Search, filters, and ingredient-based search
- Cooking Mode (step-by-step) with built-in timer
- Shopping list, favorites, and meal planner skeleton
- Ads, sponsored content, and affiliate slots
- Supabase Postgres + Storage friendly

## Deployment

Deploy to Vercel. Configure environment variables in Vercel dashboard.
